SSR配置

2020-03-30 1 min read # 干货

前提：已经能科学上网
谷歌访问助手
Astar vpn
......

打开github上一个仓库：
	https://github.com/shadowsocks/shadowsocks-windows/releases/tag/4.1.9.2
	
下载对应的最新的版本

1）将下载的两个压缩包进行解压；
2）将下图左侧文件拷进右侧文件夹，并双击运行
3）安装过程中出现问题，请安装补丁，具体参考：https://github.com/shadowsocks/shadowsocks-windows
需要安装 .NET Framework 4.7.2 和 Microsoft Visual C++ 2015 Redistributable (x86)

打开免费节点网站：
	https://free-ss.site/
右键windows桌面右下角的小飞机--->选择服务器--->扫描屏幕二维码--->确定
在搜索引擎输入谷歌和YouTube来确认
